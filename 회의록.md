# 우리아가(U-AGA) 프로젝트 회의록

## 2026-07-17: 교류 게시판(자랑하기 갤러리) 추가 논의
- **안건**: 메인 페이지 좌측 하단(체크 표시 영역)에 '교류 게시판' 추가 구현 방안 논의.
- **현재 상태**: Firebase가 연동되어 있는 상태.
- **논의 내용**:
  - Firebase Firestore(텍스트 데이터)를 활용하여 서버 구축 없이 게시판 개발 가능.

## 2026-07-17: 교류 게시판 요구사항 확정 (대표님 지시사항)
1. **권한**: 익명 게시판 (로그인 없이 작성 가능)
2. **콘텐츠 형식**: 기본적으로 텍스트(글씨) 위주로 작성하되, **선택적으로 작은 사진 1장을 첨부**할 수 있도록 기능 추가 (Firebase Firestore + Storage 함께 사용).
3. **디자인 UI**: 기존 'PARTY DIRECTORY' 리스트 형태(다크 테마 둥근 직사각형)를 그대로 차용하되, 사진을 첨부한 글의 경우 리스트 UI 어딘가에 **작은 썸네일(미리보기)**이 시각적으로 예쁘게 표시되도록 디자인 디테일 추가.

## 2026-07-17: 교류 게시판 기능 변경 (사진 첨부 제거)
- **이슈**: Firebase Storage 정책 변경으로 사진 저장소 생성 시 요금제 업그레이드(신용카드 등록) 필요.
- **대표님 지시사항**: 요금제 업그레이드 없이 무료로 사용하기 위해 **사진 첨부 기능을 제거하고 텍스트 전용 게시판(B안)으로 변경**.
- **조치 결과**: 코드에서 사진 첨부 버튼 및 이미지 업로드 로직 완전 제거 완료. (이제 Firestore만 사용하므로 카드 등록 없이 평생 무료 사용 가능)
